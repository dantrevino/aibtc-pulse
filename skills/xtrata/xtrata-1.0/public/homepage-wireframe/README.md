# Homepage Wireframe Draft

Location: `PUBLIC/homepage-wireframe/`

This wireframe is a draft for a simplified public homepage aimed at first-time builders and creators.

## What is included

- Wallet connect button in the header (available on the page at all times).
- Two visual directions in one prototype:
  - `Creator-first`
  - `Protocol-first`
- Top-of-page `grid + preview` flow.
- Quick buy action in preview for listed items only.
- Simplified inscription module with the core 3-step flow:
  - Begin
  - Upload batches
  - Seal
- Focused starter docs cards:
  - How inscriptions work
  - Artist collection access
  - Fee expectations
  - Safety checklist
- Direct link to `Workspace` (old public page target naming).

## How to review

Open `PUBLIC/homepage-wireframe/index.html` in your browser.

## Notes

- This is a design and interaction wireframe only.
- It is intentionally self-contained and not yet wired to live app/state modules.
- `/workspace` is used as the target label path in this draft and can be remapped during implementation.

# Metrics and KPIs

## Awareness
- Impressions per week
- Reach per week
- Profile visits

## Engagement
- Saves / shares per post
- Comment rate
- Avg watch time (video)

## Conversion
- Docs page views
- Viewer sessions
- Mint attempts
- Integration requests

## Weekly reporting template
- Top 3 posts by reach
- Top 3 posts by engagement
- Top 3 traffic sources
- Notes / experiments to repeat

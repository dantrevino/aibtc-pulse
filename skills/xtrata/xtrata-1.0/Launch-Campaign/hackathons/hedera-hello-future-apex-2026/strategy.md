# Hedera Hello Future Apex Hackathon 2026 Strategy

## Event snapshot
- Window: starts 2026-02-17, ~5 weeks.
- Prize framing observed: up to $250,000 pool, with direct challenge prizes
  called out separately in partner listings.
- Format: global, virtual, multi-track.
- Confidence: high.

## Recommended entry
Build **Audionals AgentPay for Hedera** (not a full Xtrata port).

## Why this is the best fit
- Hedera challenge language emphasizes real-world utility, agentic workflows,
  and microtransactions.
- Audionals can showcase direct end-user value quickly.
- Xtrata can still be referenced as the permanent provenance layer (hash/source
  of truth) while Hedera handles fast payment and coordination flows.

## Product concept
AI agents can legally license and pay for audio snippets in real time.
- Rights fingerprints and provenance anchors map to Xtrata content IDs.
- Usage events stream through Hedera services.
- Micropayments settle instantly per use, per creator split policy.

## 60-second pitch
Audionals AgentPay turns creator licensing into machine-speed commerce. Every AI
agent action that consumes media triggers a verifiable usage event and a
settled micro-payment path on Hedera. Xtrata provides permanent provenance and
deterministic media identity, while Hedera provides throughput and low-latency
settlement. The result is a production-feeling rights and payments rail for the
agent economy.

## Hackathon scope
Must-have:
- Wallet-connected licensing flow.
- Usage event log + creator payout split logic.
- Demo with two creators and one agent-style consumer flow.

Stretch:
- Dynamic pricing by usage type.
- Dashboard for payout reconciliation and usage history.

## Submission checklist
- Confirm final track selection on launch day.
- Include a clear live demo path and one-click replay script.
- Show explicit problem -> on-chain action -> user outcome loop.

# Protocol Primitives

This folder tracks long-lived recursive building blocks for Xtrata.

## Directory Map

- `numerical-anchors/` first-256 numerical anchor program, docs, and delivery plans
- `math-symbols/` symbolic and mathematical constants
- `identity/` identity and ownership primitives
- `token-logic/` reusable token and payout logic primitives
- `fun-experiments/` mythic/experimental anchor concepts
- `reserved-slots/` slots intentionally held for future governance decisions

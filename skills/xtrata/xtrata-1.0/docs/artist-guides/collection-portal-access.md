# Collection Portal Access

The collection portal (`/manage`) is for approved artist and builder wallets.
If you want to launch collections, request access first.

## How to request access

1. Prepare the wallet address you will use in the portal.
2. Share a short project summary (artist drop, builder app, timeline, expected collection format).
3. Contact the Xtrata team:
   - X account: `@XtrataLayers`
   - Jim.BTC: `@JimDotBTC`
4. Ask for collection portal allowlist access for your wallet.

## After access is approved

1. Open `/manage` with the approved wallet.
2. Follow the launch setup guide:
   - `docs/artist-guides/collection-launch-guide.md`
3. If you are deploying through the guided form:
   - `docs/artist-guides/collection-template-deploy-guide.md`

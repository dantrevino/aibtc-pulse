# Roadmap

## Phase 0

- define profile, contribution, and endorsement schemas
- choose a narrow set of artifact types for the first demonstration

## Phase 1

- publish sample profiles with evidence-linked work history
- build a graph-first portfolio viewer

## Phase 2

- add selective disclosure, team discovery, and endorsement weighting
- test cross-community identity portability

## Phase 3

- explore hiring, commissioning, and guild membership workflows on top of the graph

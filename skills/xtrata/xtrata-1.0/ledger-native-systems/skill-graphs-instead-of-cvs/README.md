# Skill Graphs Instead of CVs

Professional identity can be represented as a graph of work artifacts, collaborations, verified outputs, learning milestones, and endorsements rather than a static self-reported resume.

## Focus

- shift from claims to evidence-linked contribution graphs
- make collaboration history machine-readable
- support portable reputation without flattening people into one score

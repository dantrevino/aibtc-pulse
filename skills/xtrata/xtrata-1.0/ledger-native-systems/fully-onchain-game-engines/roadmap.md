# Roadmap

## Phase 0

- define runtime object types and compatibility descriptors
- identify a deterministic, low-complexity game loop for the first prototype

## Phase 1

- build a launcher that resolves one engine, one content pack, and one map graph
- publish a derivative game that reuses most of the engine graph

## Phase 2

- add score attestations, replay proofs, and modular reward logic
- test versioning policy for engine updates and game forks

## Phase 3

- explore portable identities, cross-game inventories, and open mod economics

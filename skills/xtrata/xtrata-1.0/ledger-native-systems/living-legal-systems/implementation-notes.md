# Implementation Notes

## Minimal prototype slice

- choose a narrow domain such as one policy area or model constitution
- encode a few statutes, amendments, and cases as linked objects
- render a graph showing how one active rule depends on prior sources

## Notes for future development

- keep canonical authority separate from interpretation layers
- expect text segmentation to matter more than in creative media graphs
- start with governance or charter documents before high-volume case law

## Possible first integrations

- Xtrata manifests for durable publication of constitutional or charter objects
- Stacks registries for amendment ordering and institutional publication state

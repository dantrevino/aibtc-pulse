# Roadmap

## Phase 0

- select one object category and define lifecycle event vocabulary
- map trust assumptions for physical verification and custody updates

## Phase 1

- publish a sample object graph with part and repair history
- build a timeline viewer for ownership, custody, and condition state

## Phase 2

- add title transfer, escrow, and verification workflows
- test restoration economics and maintenance incentive models

## Phase 3

- expand into institutional archives, insurance tooling, and circular-economy integrations

# Possible Stack

- canonical artifact storage: Bitcoin or Xtrata-linked lifecycle manifests and certificates
- programmable layer: Stacks contracts for title, escrow, claims, and maintenance incentives
- indexing layer: provenance timeline database, part genealogy cache, and condition status views
- optional hardware layer: NFC tags, secure elements, signed inspection devices
- app layer: collector passport, museum registrar console, insurer provenance viewer

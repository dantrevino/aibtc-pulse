# Possible Stack

- canonical artifact storage: Bitcoin inscriptions or Xtrata manifests across many object classes
- programmable layer: Stacks contracts for stewardship, funding, licensing, and collection governance
- indexing layer: federated graph search, ontology bridge services, and cache-friendly collection views
- optional storage mirrors: institutional repositories, IPFS, R2, and public archive mirrors
- app layer: cultural map explorer, curator tools, public archive portals, research interfaces

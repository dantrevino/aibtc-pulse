# Evolutionary DAO Governance

DAO governance can be modeled as a branching graph of constitutions, amendments, procedures, proposals, and execution outcomes instead of a flat sequence of votes against mutable contracts.

## Focus

- treat governance rules as versioned public objects
- make constitutional lineage inspectable
- let communities fork rule systems without losing history

# Research Notes

## Key research challenges

- preserving clear legitimacy when multiple branches claim continuity
- mapping natural-language constitutional intent onto executable rule modules
- preventing upgrade complexity from making governance unreadable to members
- handling emergency intervention without undermining branch integrity

## Open questions

- which governance primitives are reusable enough to become shared modules?
- how should treasury assets behave across constitutional forks?
- when should a rule change create a new branch versus updating the active one?

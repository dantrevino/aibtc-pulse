# Roadmap

## Phase 0

- define branch manifest and amendment object schemas
- identify a minimal rule set for the first constitutional graph

## Phase 1

- prototype branch-aware proposal history and constitutional diffing
- publish one forked branch from a common origin

## Phase 2

- connect branch state to executable voting and treasury contracts
- explore governance module reuse across multiple communities

## Phase 3

- test branch migration tooling, legitimacy signaling, and comparative governance analytics

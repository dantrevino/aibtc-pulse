# Executable Knowledge Objects

Scientific and research artifacts can be represented as graphs of claims, datasets, code, experiment runs, notebooks, reviews, and revisions instead of static PDF endpoints.

## Focus

- make research reproducible by default
- preserve the lineage between claims and evidence
- let later work reference exact experiments, not vague citations

## Scaffold

- `concept.md`
- `architecture.md`
- `protocol-design.md`
- `implementation-notes.md`
- `possible-stack.md`
- `research-notes.md`
- `roadmap.md`

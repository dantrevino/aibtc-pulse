# Roadmap

## Phase 0

- define claim, method, and evidence object vocabulary
- choose a narrow domain for the first demonstration

## Phase 1

- publish one executable knowledge object with linked evidence
- ship a graph viewer for claim-to-evidence traversal

## Phase 2

- add review, replication, and amendment workflows
- test incentive designs for verification and correction

## Phase 3

- expand into field-specific schemas and cross-institution collaboration patterns

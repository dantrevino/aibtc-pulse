# Possible Stack

- canonical artifact storage: Bitcoin inscriptions for manifests, notebooks, reviews, and small datasets
- programmable layer: Stacks contracts for registries, replication rewards, and publication status
- indexing layer: claim graph store, execution metadata index, and reproducibility status cache
- optional storage mirrors: object storage or IPFS for large datasets and container artifacts
- app layer: executable paper viewer, replication queue, peer review dashboard

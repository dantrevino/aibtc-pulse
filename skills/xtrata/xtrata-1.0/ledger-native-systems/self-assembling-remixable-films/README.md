# Self-Assembling / Remixable Films

Films can be represented as timeline graphs whose nodes include scenes, shots, sound cues, edit decisions, subtitle packs, VFX layers, and alternate cuts.

## Focus

- make film lineage visible at the scene and edit level
- enable alternate cuts without duplicating entire productions
- support rights and revenue flows across complex collaborative media

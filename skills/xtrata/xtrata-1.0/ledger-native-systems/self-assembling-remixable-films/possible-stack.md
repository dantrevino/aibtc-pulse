# Possible Stack

- canonical artifact storage: Bitcoin inscriptions for cut manifests, subtitles, and key scene assets
- programmable layer: Stacks contracts for edition rights, split routing, and release registration
- indexing layer: timeline graph resolver, cut diff engine, and scene reuse cache
- optional storage mirrors: object storage or streaming CDN for high-bitrate media payloads
- app layer: timeline explorer, alternate-cut player, restoration archive viewer

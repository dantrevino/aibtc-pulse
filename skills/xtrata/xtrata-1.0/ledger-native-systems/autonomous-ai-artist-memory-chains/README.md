# Autonomous AI Artist Memory Chains

AI creative agents can be treated as lineage systems: prompts, datasets, model versions, critiques, tool settings, and outputs all become linked objects in a memory graph.

## Focus

- make AI creative evolution inspectable
- preserve prompt, model, and dataset provenance
- turn agent memory into a reusable cultural artifact

# Possible Stack

- canonical artifact storage: Bitcoin inscriptions for prompts, manifests, and output artifacts
- programmable layer: Stacks contracts for agent IDs, rights, payout rules, and permissions
- indexing layer: lineage graph index plus vector or semantic retrieval layer for memory search
- optional storage mirrors: encrypted blobs or object storage for large private context artifacts
- app layer: agent studio, provenance viewer, branch diff tool, licensing dashboard

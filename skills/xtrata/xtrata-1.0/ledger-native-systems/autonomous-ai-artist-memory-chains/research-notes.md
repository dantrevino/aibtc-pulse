# Research Notes

## Key research challenges

- balancing provenance with privacy for prompts, datasets, and user inputs
- representing proprietary model APIs in a durable, comparable way
- detecting fabricated lineage claims when parts of inference happen off-chain
- pricing reusable prompt and critique assets without over-financializing every run

## Open questions

- what level of run detail is enough for cultural auditability?
- how should rights flow when a human edits or curates an AI output heavily?
- can verifiable compute receipts be attached without making the system unusably heavy?

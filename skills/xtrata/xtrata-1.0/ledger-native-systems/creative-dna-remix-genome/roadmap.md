# Roadmap

## Phase 0

- define the manifest schema and dependency edge types
- map royalty models for direct and indirect ancestors

## Phase 1

- prototype a lineage-aware mint and viewer flow
- test graph reconstruction against a small remix catalog

## Phase 2

- add rights templates, split settlement, and descendant search
- evaluate builder APIs for third-party remix tools

## Phase 3

- expand from audio-first examples to visual, film, and code-native compositions

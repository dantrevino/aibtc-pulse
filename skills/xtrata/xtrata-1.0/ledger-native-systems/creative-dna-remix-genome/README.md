# Creative DNA / Remix Genome

Creative works become dependency graphs of samples, stems, scenes, layers, prompts, or code fragments. New works are assembled by referencing reusable components instead of copying opaque final files.

## Focus

- preserve provenance across remix chains
- route attribution and royalties upstream automatically
- treat "composition" as a graph operation, not just a rendered export
